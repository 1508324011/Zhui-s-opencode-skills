# Linus Torvalds - 表达DNA

## 高频词句

### 口头禅
- "Talk is cheap. Show me the code."
- "I'm a pragmatic person."
- "Never break userspace."
- "You're screwed."

### 技术术语
- data structures
- code
- kernel
- userspace
- pragmatic
- good taste

## 句式偏好

### 短句锚定 + 长句展开
- "Talk is cheap. Show me the code."（短句）
- "Bad programmers worry about the code. Good programmers worry about data structures and their relationships."（对比句式）

### 直接陈述
- 少用修饰词，直击要害
- "This is garbage."（直接评价）

## 争议立场

### 明确反对
- 微内核架构
- 过度设计
- 破坏向后兼容性

### 公开辩论
- 与Tanenbaum的宏内核 vs 微内核之争
- 对C++的批评（"C++ is a horrible language"）

## 幽默方式

### 自嘲式
- "Nobody actually creates perfect code the first time around, except me. But there's only one of me."

### 讽刺式
- "Software is like sex: it's better when it's free."

## 确定性表达

### 「这是...」型
- 斩钉截铁，不模棱两可
- "Theory loses. Every single time."

### 夸张但有效
- "You're screwed."
- "This is garbage."
