# Linus Torvalds - 时间线

## 完整时间线

| 年份 | 事件 | 意义 |
|------|------|------|
| 1969 | 出生于芬兰赫尔辛基 | - |
| 1988 | 进入赫尔辛基大学 | 开始接触Unix和C语言 |
| 1991 | 发布Linux内核0.01 | 开源运动的重要里程碑 |
| 1992 | 采用GPL许可证 | 确保代码始终开源 |
| 1994 | Linux 1.0发布 | 第一个稳定版本 |
| 1996 | 获得硕士学位 | 学业完成，全心投入Linux |
| 1997 | 移居美国 | 加入Transmeta公司 |
| 2003 | 加入OSDL | 全职维护Linux内核 |
| 2005 | 创建Git | 解决Linux内核版本控制问题 |
| 2007 | Linux Foundation成立 | 继续领导内核开发 |
| 2012 | 获得千禧技术奖 | 开源贡献获得国际认可 |
| 2016 | TED演讲 | 公开阐述"好品味"哲学 |
| 2018 | 短暂休假 | 反思沟通风格 |
| 2020至今 | 继续领导Linux开发 | 持续贡献 |

## 关键里程碑

### 技术里程碑
- 1991: Linux 0.01 发布
- 1994: Linux 1.0 发布
- 2005: Git 创建

### 思想转折点
- 1991: 从Minix用户到Linux创造者
- 2005: 从BitKeeper用户到Git创造者
- 2016: TED演讲，系统阐述编程哲学

## 最近12个月动态 (2025-2026)

- 继续领导Linux内核开发
- 参与Git的维护和演进
- 技术访谈和公开演讲
